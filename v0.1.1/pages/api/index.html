Internals

Query representation

YeSQL.SQLQuery — Type
struct SQLQuery{O<:YeSQL.SQLOperation}

SQL query with fields extracted from a .sql file or string

Fields

  • name::Symbol

    Query name

  • query::String

    Processed query string

  • docstring::String

    Docstring

  • args::Vector{Symbol}

    Argument list

  • operation::YeSQL.SQLOperation

    SQLOperation derived from the operation suffix

Query syntax

Each query should be preceded by comment lines defining the query name and operation type, and an optional docstring:

-- name: <name><operation_suffix>
-- <docstring>  (optional)
<query text>
source

SQL Operations

These empty types are used for dispatch when executing a query. When queries are loaded the query name suffix is used to determine the appropriate operation.
